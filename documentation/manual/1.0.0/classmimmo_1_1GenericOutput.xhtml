Detailed Description

GenericOutput is the class that write the an output on file.

Date
09/feb/2016
Authors
Rocco Arpa
Edoardo Lombardi

GenericOutput is derived from BaseManipulation class. It uses and it writes the base members input.

WARNING AbsorbSectionXML, FlushSectionXML not coded! It's using baseManipulation default;
