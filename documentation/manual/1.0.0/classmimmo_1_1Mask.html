Detailed Description

Mask is the class that applies a geometrical masking filter to a set of data.

Date
09/feb/2016
Authors
Rocco Arpa
Edoardo Lombardi

Mask is derived from BaseManipulation class. It uses the members m_coords as values to be compared with the thresholds fixed by the user for each coordinate. The flags into m_inside define, for each coordinate, if the object applies the masking inside or outside the thersholds.

WARNING AbsorbSectionXML, FlushSectionXML not coded! It's using baseManipulation default;
