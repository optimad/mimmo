 Utilities to create port connections between executable blocks.Here are collected enums and methods to create connections between blocks throughout established ports. All the available port types are collected in a special enum PortType, istantiated through BETTER_ENUM macro, and each type reports a specific TAG in order to be retracked in each moment for compatibility checking purposes.
