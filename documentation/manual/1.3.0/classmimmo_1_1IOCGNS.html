Detailed Description

IOCGNS is the class to import/ export an unstructured volume mesh written in CNGS.

The object provides an interface to retrieve volume mesh and its boundary surfaces and importing them in mimmo's native Data structures of type MimmoObject

The class is in beta version and has the following limitations:

  • Only unstructured mesh with single base supported.
  • Singular of multi-zone meshes are supported in reading native cgns.
  • Singular zone meshes are supported in writing cgns format.
  • Multi-zone meshes cgns writing will be supported in future, but not yet available.
  • Data attached to boundary conditions patches are not read/written, except for their names and their cgns type.
  • CGNS meshes exported from Pointwise16 and StarCCM++ are still unreadable with the current class. Errors are known and will be fixed in later versions.
  • Reading/writing partitioned mesh in cgns format is not yet available.

Dependencies : cgns libraries.

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation:

  • ClassName: name of the class as mimmo.IOCGNS;
  • Priority: uint marking priority in multi-chain execution;

Proper of the class:

  • IOCGNS_Mode: working mode of the class, see IOCGNS_Mode enum.
  • Dir: directory path for reading/writing;
  • Filename: name of file to read/write without .cgns/dump tag;
  • WriteInfo: boolean (1/0) write on file zoneNames, bcNames, either in reading and writing mode. The save directory path is specified with Dir.
  • WriteFormat: writing format supported by the class, see IOCGNS_WriteType enum
  • WriteMultiZone: 0- write single zone, 1- write multizone(if multi zone are available in the mesh).

Geometry has to be mandatorily read or passed through port.

◆ setFilename()

void mimmo::IOCGNS::setFilename ( const std::string &  filename)

It sets the filename without tag (.***) for IO operation. Given a target name "test", for each mode ew will have:

  • READ : read from Dir the cgns file test.cgns (MPI with 0-rank only).
  • RESTORE : read from Dir the dump file test.xxx.dump.
  • WRITE : write to Dir the cgns file test.cgns, containing the inner mesh (MPI with 0-rank only).
  • DUMP : write to Dir the dump file test.xxx.dump, containing the inner mesh.

In case setWriteOnFileMeshInfo is set to true (or WriteInfo = 1 in xml) the mesh info file test_MeshInfo.dat will be written on Dir, for READ and WRITE mode only .

Parameters
[in]filenameName of target file.
Examples:
iocgns_example_00001.cpp, and iocgns_example_00002.cpp.
