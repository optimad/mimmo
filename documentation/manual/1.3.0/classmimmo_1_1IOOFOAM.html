Detailed Description

IOOFOAM is the class to import/export an OpenFOAM volume mesh alongside its boundary information.

The class is derived from MimmoFvMesh interface.

It works as reader and writer. Its mode types are summarized in the following better enum IOOFMode list:

  • READ : import bulk and boundary mesh from an OpenFOAM case
  • WRITE : export bulk and boundary mesh to an OpenFOAM case from scratch
  • WRITEPOINTSONLY : export only bulk mesh points coordinates to a pre-existent and compatible OpenFOAM case.

Warning: WRITE mode is not available yet. It triggers the WRITEPOINTSONLY mode, which is a temporary mode to write only the coordinates of OpenFOAM mesh points.

Dependencies : OpenFOAM libraries (tested with OpenFOAM 2.4.x version).

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation/MimmoFvMesh:

  • ClassName: name of the class as mimmo.IOOFOAM;
  • Priority: uint marking priority in multi-chain execution;

Proper of the class:

  • IOMode: activate mode of the class: READ, WRITE, WRITEPOINTSONLY;
  • Dir: path to the current OpenFOAM mesh for reading/writing purposes;
  • Overwrite: option valid only in WRITEPOINTSONLY mode: if 1-true overwrite points in the current OpenFoam case time of the mesh at WriteDir. If 0-false (DEFAULT) save them in a newly created case time at current time + 1;

In case of writing mode Geometries have to be mandatorily passed by port.

mimmo::IOOFOAM::IOOFOAM ( std::unique_ptr< MimmoObject > &  bulk,
std::unique_ptr< MimmoObject > &  boundary,
int  type = IOOFMode::WRITE 
)

Custom constructor. Pass bulk and boundary of the mesh. The ownership of the argument will be transferred to the internal members of the class. Please note, The class admits only the following combinations as bulk/boundary pair:

  • bulk MimmoObject of type 2-Volume and boundary MimmoObject of type 1-Surface
  • bulk MimmoObject of type 1-Surface and boundary MimmoObject of type 4-3DCurve.

The class is meant for writing modes WRITE and WRITEPOINTSONLY. If a READ mode is passed in argument, it throws an error.

Parameters
[in]bulkunique pointer to the bulk MimmoObject
[in]boundaryunique pointer to the boundary MimmoObject
[in]typeint from enum IOOFMode, for class mode: WRITE, WRITEPOINTSONLY. Default is WRITE.
