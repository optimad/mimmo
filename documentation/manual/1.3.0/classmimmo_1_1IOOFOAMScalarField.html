Detailed Description

IOOFOAM_Kernel specialization to import/export an OpenFOAM scalar field defined on a volume mesh and/or the related boundary fields.

The class presumes mesh is already read by the User with a IOOFOAM specialized class. Thus it requires a valid link for both bulk and boundary geometry in input (M_GEOMOFOAM ports), coherent with the mesh defined the in OpenFoam directory where the field is defined.

WARNING: WRITE mode for fields non available yet.

The xml available parameters, sections and subsections are the following :

  • ClassName: name of the class as mimmo.IOOFOAMScalarField;
  • Priority: uint marking priority in multi-chain execution;
  • IOMode: activate mode of the class: READ, WRITE;
  • Dir: path to the current OpenFOAM mesh for reading/writing purposes;
  • FieldName: name of the OpenFOAM field for reading/writing purposes;

Geometries must be passed by ports.
