Detailed Description

template<std::size_t NCOMP>
class mimmo::PropagateField< NCOMP >

Executable block that provides the computation of a field over a 3D volume mesh. The field is calculated solving a Laplacian problem over the mesh with given Dirichlet boundary conditions.

Class/BaseManipulation Object managing field defined on the boundaries of a 3D volume mesh. It uses MimmoObject informations as input geometry. The key to handle with constraints is an explicit calculation of the solution of a Laplacian problem. In Laplacian solver the weights used to compute the stencil are of the form: w(x) = (1/d(x))^n, where d(x) is the distance of the neighbour point to the center point of the stencil and n is the weight factor, that can be tuned by the User. Quality of the mesh during deformation is controlled by introducing an artificial diffusivity D (DUMPING) in the laplacian calculation: div( D*grad(Phi)) = 0. Such diffusivity can be defined alternatively as variable with distance from a prescribed deforming surface (dumping surface), or with cell volumes distribution, modulated with distance from the dumping surface. In the first case, cells more distant from dumping surfaces cells are forced to move more than nearer ones, in the second case, bigger volume cell far from dumping surface are forced to move more. Result field is stored in m_field member and returned as data field through ports.

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation:

  • ClassName : name of the class as mimmo.PropagateField;
  • Priority : uint marking priority in multi-chain execution;
  • PlotInExecution : plot optional results in execution;
  • OutputPlot : path to store optional results.

Proper of the class:

  • Solver : 1-true use direct Laplacian solver, 0-false use iterative Smoother (default 0);
  • WeightConstant : coefficient used to get weights of the stencil points in function of distance (1.0 default);
  • Dumping : 1-true activate dumping control, 0-false deactivate it.
  • DumpingType : 0- distance control, 1-volume control.
  • DumpingInnerDistance : inner limit of dumping function eta, if dumping is active;
  • DumpingOuterDistance : outer limit of dumping function eta, if dumping is active;
  • DecayFactor : exponent to modulate dumping function (as power of), if dumping is active
  • SmoothingSteps : number of steps the Smoother solver need to perform (1 default);
  • Convergence : convergence flag for smoothing solver;
  • Tolerance : convergence tolerance for laplacian smoothing and direct solver; Geometry, boundary surfaces, boundary condition values for the target geometry have to be mandatorily passed through ports.
