Detailed Description

template<std::size_t NCOMP>
class mimmo::PropagateField< NCOMP >

Executable block that provides the computation of a field over a 3D volume mesh. The field is calculated solving a Laplacian problem over the mesh with given boundary conditions.

Class/BaseManipulation Object managing field defined on the boundaries of a 3D volume mesh. It uses MimmoObject informations as input geometry. The key to handle with constraints is an explicit calculation of the solution of a Laplacian problem. The Laplacian solver employs a node-based scheme (GRAPHLAPLACIAN). Quality of the mesh during deformation is controlled by introducing an artificial diffusivity D (DUMPING) in the laplacian calculation: \(\nabla\cdot( D\nabla\Phi) = 0\). Such diffusivity can be defined alternatively as variable with distance from a prescribed deforming surface (dumping surface), or with cell volumes distribution, modulated with distance from the dumping surface. In the first case, cells more distant from dumping surfaces cells are forced to move more than nearer ones, in the second case, bigger volume cell far from dumping surface are forced to move more.

WARNING: dumping is an experimental feature, subject of still ongoing investigations. Sometimes it can lead to unpredicatable results. Use it carefully and at your own risk.

Result field is stored in m_field member and returned as data field through ports.

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation:

  • ClassName : name of the class as mimmo.PropagateField;
  • Priority : uint marking priority in multi-chain execution;
  • PlotInExecution : plot optional results in execution;
  • OutputPlot : path to store optional results.

Proper of the class:

  • Dumping : 1-true activate dumping control, 0-false deactivate it.
  • DumpingType : 0- distance control, 1-volume control.
  • DumpingInnerDistance : inner limit of dumping function eta, if dumping is active;
  • DumpingOuterDistance : outer limit of dumping function eta, if dumping is active;
  • DecayFactor : exponent to modulate dumping function (as power of), if dumping is active
  • Tolerance : convergence tolerance for laplacian solver;
  • UpdateThres : lower threshold to internally mark cells whose field norm is above its value, for update purposes
  • ForceDirichlet : 1 -reforce Dirichlet on Boundaries, 0-do nothing. Meaningful in Method 1- Finite Volume
  • Method : 0 - GraphLaplacian(on mesh nodes)
  • Print : print solver debug information, Active only in if MIMMO_ENABLE_MPI is enabled in compilation.

Geometry, boundary surfaces, boundary condition values for the target geometry have to be mandatorily passed through ports.
