Detailed Description

GenericInput is the class that set the initialization of a generic input data.

GenericInput is derived from BaseManipulation class. GenericInput can read the input from a data file (unformatted/csv) or it can be set by using setInput methods. When reading, the GenericInput object recognizes the type of data and it adapts the reading method in function of the output port used to build link (pin) with other objects.

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation:

  • ClassName: name of the class as mimmo.GenericInput;
  • Priority: uint marking priority in multi-chain execution;

Proper of the class:

  • ReadFromFile: 0/1 set class to read from a file;
  • CSV: 0/1 set class to read a CSV format;
  • ReadDir: path to your current file data;
  • Filename: path to your current file data.
