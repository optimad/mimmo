Detailed Description

Abstract Interface class for Elementary Shape Representation.

Date
03/01/2015
Authors
Edoardo Lombardi
Arpa Rocco

Interface class for Volumetric Core Element, suitable for interaction with Data Structure stored in a MimmoObject class. Object orientation in 3D space can be externally manipulated with dedicated transformation blocks. Class internally implement transformation to/from local sdr to/from world sdr, that can be used in derived objects from it. Class works with three reference systems: 1) Global Absolute SDR: is the external World reference system 2) Local Relative SDR: is the local reference system, not affected by Rigid Transformations as RotoTranslations or Scalings 3) basic SDR: local system remapping to unitary cube, not accounting of the shape type.
