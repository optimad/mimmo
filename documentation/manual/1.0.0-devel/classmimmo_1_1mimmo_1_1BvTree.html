Detailed Description

Bv-Tree is the class to manage a Bounding Volume Hierarchy tree of a bitpit patch.

Date
20/may/2016
Authors
Edoardo Lombardi

A Bv-Tree is composed by its nodes and the elements related to each node. A node has two possible child given by splitting in two sub-volume its bounding box. The bounding box of each node is computed as the bounding volume of all the simplex (elements) of the geometry contained in the box. In order to split the box the average of the centroids of the elements is computede and then the volume is divided by using the plane normal to the direction of maximum span of the centroids. Each node has the information about many elements are contained in its bounding volume and the first and last index of them as stored in the structure elements. The elements are ordered in function of the coordinates of their centroid during the construction of the tree. The leaf nodes can be have a number of elements in their bounding volume greater then one.
