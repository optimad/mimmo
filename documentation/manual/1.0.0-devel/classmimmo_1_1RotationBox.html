Detailed Description

RotationBox is the class that applies a rotation to a given reference system.

Date
09/feb/2016
Authors
Rocco Arpa
Edoardo Lombardi

The used parameters are the rotation value and the direction and the origin of the rotation axis.

WARNING AbsorbSectionXML, FlushSectionXML not coded! It's using baseManipulation default;
