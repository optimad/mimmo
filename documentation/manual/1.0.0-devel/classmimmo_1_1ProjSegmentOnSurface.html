Detailed Description

ProjSegmentOnSurface is an executable block class capable of projecting elemental 1D segments on a 3D surface mesh defined by a MimmoObject.

Date
30/nov/2016
Authors
Rocco Arpa

ProjSegmentOnSurface project a given segment on a given 3D surface mesh and return a discrete 3D curve mesh in MimmoObject container
