mimmo

Surface manipulation and mesh morphing library

mimmo overview

mimmo is a open source C++ library for mesh manipulation and morphing developed by Optimad Engineering Srl. The goal of mimmo is to provide mesh manipulation objects, input/output interfaces and morphing wrappers. The library is designed as a block diagram framework where each object can be linked with others in an execution chain.

mimic

mimmo can be used as a building block for developing mesh morphing software.

Optimad Engineering Srl is using mimmo to develop mimic, a mesh morphing software specifically designed for being seamlessly integrated in industrial optimization workflows. In addition to the capabilities provided by mimmo, mimic implements advanced advanced features like morphing of volume meshes, continuity constratins and an OpenFOAM input/output interface.
