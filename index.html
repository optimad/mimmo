MiMMO

Manipulation and Morphing for Meshes Open Source

A quick look at MiMMO

MiMMO is a open source C++ library for mesh manipulation and morphing.

MiMMO is developed and maintained by OPTIMAD Engineering Srl and is distributed under the GNU Lesser General Public License v3.0.

The framework consists of a number of manipulation objects, input/output interfaces, morphing wrappers and other tools.
The library is designed as a block diagram framework where each object can be linked with others in an execution chain.

Inside MiMMO some basic manipulation objects are provided, as well as some wrappers to fundamental deformations and other tools convenient to surface and mesh morphing.
The implementation of customized blocks is straightforward by deriving the own object class from the base class defined in MiMMO.

The library allows to import/export several CAD format (stl, vtu, cgns and others). The permitted formats are listed in the documentation of input/output classes.

MiMMO is based on bitpit, a C++ library developed by OPTIMAD. bitpit supplies basic tools to manage surface and volume meshes.

MiMMO is the base of MIMIC, a commercial software developed and distributed by OPTIMAD aimed to surface and volume mesh morphing for industrial applications and shape optimization. MIMIC, compared to MiMMO, provides advanced tools and interfaces, e.g. volume mesh morphing, continuity constratins definition, OpenFOAM input/output interfaces.

Conversely to MiMMO, MIMIC provides even a graphical user interface.

MiMMO is part of PASO the platform for automatic shape optimization developed by OPTIMAD.
